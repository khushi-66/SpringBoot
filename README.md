# Spring Boot Learning Repository

This repository contains a collection of **Spring Boot practice projects and concept implementations** created while learning backend development with Java and Spring Boot.

The goal of this repository is to explore and understand core Spring Boot features, REST API development, and backend architecture through hands-on practice.

---

## Key Concepts Covered

* Spring Boot Fundamentals
* REST API Development
* MVC Architecture
* Dependency Injection
* CRUD Operations
* MicroService Architecture
* Spring Data JPA
* Layered Architecture (Controller, Service, Repository)
* Application Configuration

---

## Tech Stack

* Java
* Spring Boot
* Microservices
* Thymeleaf
* RestAPI
* Spring Security
* Spring MVC
* Spring Data JPA
* Hibernate
* Maven
* MySQL

---

## Repository Structure

Each directory represents a small practice project or a concept-based implementation.

Example:

 * p ->   Core Spring and Spring Data JPA 
 * r ->   RestAPI and spring mvc
 *  M ->   MicroServices

---

## Purpose of this Repository

This repository is maintained for:

* Practicing Spring Boot concepts
* Experimenting with backend implementations
* Strengthening Java backend development skills

Production-level and portfolio-ready projects are maintained in **separate repositories**.
